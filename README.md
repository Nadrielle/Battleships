## Installation and Setup Instructions

You will need `node` and `npm` installed globally on your machine.  

Clone down this repository:
`git clone https://github.com/Nadrielle/Battleships.git`

Installation:

`npm install`

To Start Server:

`npm start`

To Visit App:

`localhost:3000/`
